# denkr 0.1.0

## Summary

First named `0.1.0` Android beta release for denkr, focused on making the assistant feel more stable, inspectable, and ready for continued private beta testing.

## What changed

- Hardened backup behavior and Android beta install handling
- Updated legal notices, developer context, and prompt assembly with app version information
- Improved chat bubble markdown rendering and transcript persistence
- Added content-based text direction for chat messages
- Added Google OAuth support for skills
- Implemented Gemini native web search grounding and updated related policies
- Improved web fetch support for reading X.com status URLs
- Refined theme seeds, bundled Thmanyah fonts, and expanded font preset documentation

## How to use it

- Download the APK from the matching GitHub Release asset
- Install it on Android from the downloaded file
- If Android blocks installation, allow installs from the browser or file manager you used for the download
- In denkr, open Settings -> App Update to check whether a newer beta build exists

## Known issues

- Android beta updates are manual installs outside Google Play
- APK install confirmation still happens through the Android system installer
- iOS is not wired into this release channel yet
