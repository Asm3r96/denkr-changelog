# denkr 0.2.0

## Summary

Temporary Android beta release used to verify direct in-app update checks from Settings.

## What changed

- Publishes a higher Android beta version so installed `0.1.0` builds can detect an available update
- Uses the direct APK release asset path expected by the app update screen
- Keeps the same app behavior as the current local beta test build

## How to use it

- Open denkr Settings -> App Update
- Confirm the app sees version `0.2.0`
- Tap Download and install the APK from Android's installer prompt

## Known issues

- This is a temporary test release and can be deleted after update verification
- Android beta updates are manual installs outside Google Play
- APK install confirmation still happens through the Android system installer
