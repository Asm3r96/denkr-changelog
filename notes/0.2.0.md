# denkr 0.2.0

## Summary

First public Android beta release of denkr.

## What changed

- This is the first beta build shared through the public denkr release channel
- The app can check this channel from Settings -> App Update
- The APK is available as a direct GitHub Release download

## How to use it

- Download the APK from this release
- Install it on Android from the downloaded file
- If Android blocks installation, allow installs from the browser or file manager you used for the download

## Known issues

- Android beta updates are manual installs outside Google Play
- APK install confirmation still happens through the Android system installer
- iOS is not wired into this release channel yet
